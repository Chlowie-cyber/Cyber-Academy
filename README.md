# Cyber Academy

Cyber Academy is a modern cybersecurity learning platform built with React, TypeScript, Vite, and Tailwind CSS. It guides users through interactive learning units, tracks progress, rewards XP, and helps connect training to cybersecurity career pathways.

**Live Demo:** [https://cyber-academy-project.vercel.app/]

## Features

- Interactive cybersecurity learning modules
- Progress tracking across multiple units
- XP and streak-based motivation system
- Career matrix to explore cybersecurity roles
- Recommended virtual simulations and portfolio-building activities
- Clean, modern UI designed for learning and engagement

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- PostCSS

## Getting Started

1. Clone the repository

\`\`\`bash
git clone https://github.com/Chlowie-cyber/Cyber-Academy.git
\`\`\`

2. Navigate into the project folder

\`\`\`bash
cd Cyber-Academy/project
\`\`\`

3. Install dependencies

\`\`\`bash
npm install
\`\`\`

4. Start the development server

\`\`\`bash
npm run dev
\`\`\`

5. Open your browser and visit the local URL shown by Vite.

## Project Structure

\`\`\`
project/
  src/
    components/     # Reusable UI components
    data/           # Curriculum and learning content
    hooks/          # App state and logic
    App.tsx         # Main application view
    main.tsx        # Entry point
\`\`\`

## Usage

- Start learning from the main training dashboard
- Complete units to earn XP and unlock new content
- Explore career pathways and recommended simulations
- Reset progress at any time from the interface
